# Clouds

Flying clouds from left to right component.

#### Example

<Canvas of={Clouds.Up} />

## Usage

```ts
import {Clouds} from "react-lovely-animations";

const Example = () => {
    return (
        <Clouds>
            <div>Just an example</div>
        </Clouds>
    );
};

export default Example;
```

#### Arguments

- **row** - A string with one of three possible values: "up", "middle" or "down". It defines the location of clouds. Default = "up".
